How color earns its place
Orange #FF6B35 — the universal accent. The only interactive color: active tab, links, primary buttons. Doubles as the urgent-attention index (the 3px rail on Needs-you rows). Used sparingly so it always means "act here."
Green — healthy motion. Running sandboxes, live sync dot, positive outcomes. Quiet, never competes with orange.
Steel — neutral info. Planned / ready states, non-urgent metadata. A supporting blue, never an accent.
Red — failure only. Failed runs, destructive confirms. Reserved so it stays legible as alarm.
